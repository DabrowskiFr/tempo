# Snake Raylib (Tempo.App)

Port de `examples/snake_tsdl.ml` avec architecture `Tempo.App`.

Controles:
- Fleches: direction
- `P`/`SPACE`: pause
- `R`: restart
- `ESC`: quitter

Build:
```bash
cd /Users/fredericdabrowski/Repos/tempo/applications/games/snake-raylib
dune build
dune exec ./main.exe
```
