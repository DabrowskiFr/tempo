# snake-raylib

## Overview

Démo simple Snake basée sur Tempo.App et rendue avec Raylib.

## Controls / Inputs

- `Fleches`: direction
- `P`/`SPACE`: pause
- `R`: restart
- `ESC`: quitter

## Launch Commands

Depuis `/Users/fredericdabrowski/Repos/tempo`:

```bash
dune exec ./applications/run -- snake-raylib
```

Direct:

```bash
dune exec ./applications/simple-demos/snake-raylib/run
```
