# cloth-raylib

## Overview

Port Raylib de `examples/cloth_tsdl.ml` (simulation masse-ressort / cloth-jelly).

- une tâche Tempo par nœud,
- ressorts structurels + diagonaux + secondaires,
- rendu dynamique des contraintes.

## Controls

- `Q` ou `ESC`: quitter

## Launch

```bash
sh applications/simple-demos/cloth-raylib/run
```
