# solar-system-raylib

## Overview

Démo simple d’un système solaire 2D avec intégration temporelle en Tempo.

## Controls / Inputs

- `SPACE` ou `P`: pause/reprise
- `ESC`: quitter

## Launch Commands

Depuis `/Users/fredericdabrowski/Repos/tempo`:

```bash
dune exec ./applications/run -- solar-system-raylib
```

Direct:

```bash
dune exec ./applications/simple-demos/solar-system-raylib/run
```

## CLI Options

- Pas d'options CLI publiques documentées.

## Dependencies

- OCaml 5.x, dune, raylib

## Headless / Reproducible Mode

- Pas de mode headless public standardisé.

## Troubleshooting

- Si animation trop rapide/lente: vérifier FPS système et charge machine.
